# video-analysis

An LLM skill that turns a video into a self-contained HTML report: keyframes with
timestamps, each paired with concise takeaways generated by transcribing the audio and
analyzing every keyframe against the transcript with a vision model.

## Usage

```bash
uv run --with openai python scripts/analyze_video.py INPUT.mp4 --topic "Acme's product"
```

Requires `ffmpeg`/`ffprobe` on PATH and an OpenAI key in `OPENAI_KEY` or `OPENAI_API_KEY`.
Output lands in `outputs/INPUT_analysis/report.html`.

See [SKILL.md](SKILL.md) for the full pipeline, tuning knobs, and gotchas.
